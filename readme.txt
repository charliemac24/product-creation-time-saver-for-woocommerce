Create templates that will help you save time in creating products.

== Description ==

Creating WooCommerce product requires patience specially when dealing with too many attributes and product info. This plugin will set these automatically for you when adding products.

**BASIC**
The basic version of this plugin will allow you to create single template only.

**PRO**
The PRO version of this plugin will allow you to create multiple templates.

Email [charlieanchetamacaraeg@gmail.com](mailto:charlieanchetamacaraeg@gmail.com) for the PRO version and customizations.

1. Navigate to Products > Product Creation Time Saver Template
2. Create a template
3. Go create new product
4. You'll see "Apply Time Saver Template" dropdown just beside the "Add new product" at the top of your screen.
5. Select your created template and product attributes, categories, tags etc will automatically setup for you.



== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Product Creation Time Saver"
3. **Install** the plugin
3. **Activate** the plugin
